ポイント連携プラグイン

独自開発したポイント管理システムにEC-CUBEを連携させるためのプラグインです。

Download this project as a .zip file Download this project as a tar.gz file

ポイント連携プラグインのインストール

ポイント連携プラグインはこのページよりtar.gz形式でダウンロードし、そのファイルをEC-CUBE管理画面のプラグイン管理よりアップロードすることでインストールできます。 現在はアップデート処理の正常動作が確認できていないため、お手数ですが、バージョンアップの際は一度削除した上で再インストールを行なって下さい。

インストール後は接続先のサーバー情報がパラメータ設定から行えます。

プラグインの動作

プラグインを有効にすると、ポイント取得、変更時にポイントシステムに連携して、EC-CUBE上のポイントを設定し直します。 購入時のポイントはEC-CUBE上でのポイント計算を使用しますが、入会時、初回購入時、購入金額累計ポイントの計算はポイントシステム上で行われます。

購入系のポイントも連携するため、注文データについても、ポイントシステム側に送信します。

将来的には、このシステムに別途開発した売上分析ツールの組み込みも予定しています。
